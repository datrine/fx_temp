

## Description

[Nest](https://github.com/nestjs/nest) Fx Trading App repository.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

Set up environment variables as described in .env.sample file

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
